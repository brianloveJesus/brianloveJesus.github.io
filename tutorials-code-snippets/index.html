Code for marching cubes blobs in C# for unity3d

- SoftBody.cs or SoftBodyMP.cs source files or SoftBodyMP.unitypackage
Code for softbodies (Jello blocks and cloth) for unity3d in C#, sorry i haven't updated it to the latest version. The mp versions use midpoint integration to increase the stability of the system at the cost of some processing time. Comments included in source code.

- VGOBlob.java - Source code to the plasma applet that is displayed above. It uses a MemoryImageSource and creates a color plasma for the background, then renders 2D blobs that appear as flares (they grow as they get closer to each other), and passes everything through one of 4 different diffraction lenses, including one that is a loaded image.

- CGFeeds Widget - A Yahoo Widget that interprets the RSS feed from CGFeeds and displays the Image for each entry, as well as transfering you to the relevant page when double clicked. It has a slideshow option as well.
